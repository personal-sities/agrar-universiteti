Kerakli bo‘limni tanlang

Bu sahifa 2 ta bo‘limdan iborat: Skript va Qabul. Ma’lumotlar Google Sheets orqali olinadi va yangilanadi.

Yangi tuzilma

Tepada bitta bosh qism chiqadi, pastida esa batafsil ma’lumotlar kartalar ko‘rinishida chiqadi. Ma’lumot qo‘shish va import qilish mumkin. Import qilishda parol so‘raladi.

Ma’lumot qo‘shish va import qilish

Import fayl JSON formatda bo‘lishi kerak. Importdan oldin parol so‘raladi.
JSON misol: [{"title":"...","summary":"...","details":"..."}]
